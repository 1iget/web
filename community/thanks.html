We'd like to thank the following individuals and organizations who contribute to the OpenSSL project.

  • Our current hosting is provided courtesy of SpaceNet AG.
  • Thanks to rsync.net for providing free backup storage.
  • Thanks to Akamai Technologies for providing free CDN and security services.
  • Thanks to Travis CI for for powering OpenSSL's continuous integration pipeline.
  • Thanks to Amazon Web Services for a grant that we are using to build better fuzzing tests.
