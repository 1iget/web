
   OpenSSL version 0.9.8v released
   ===============================

   OpenSSL - The Open Source toolkit for SSL/TLS
   http://www.openssl.org/

   The OpenSSL project team is pleased to announce the release of
   version 0.9.8v of our open source toolkit for SSL/TLS. This new
   OpenSSL version is a security and bugfix release. For a complete
   list of changes, please see

       http://www.openssl.org/source/exp/CHANGES.

   The most significant changes are:

      o Fix for ASN1 overflow bug CVE-2012-2110

   We consider OpenSSL 0.9.8v to be the best version of OpenSSL 0.9.8v
   available and we strongly recommend that users of older 0.9.8v versions
   upgrade as soon as possible. OpenSSL 0.9.8v is available for
   download via HTTP and FTP from the following master locations (you
   can find the various FTP mirrors under
   http://www.openssl.org/source/mirror.html):

     * http://www.openssl.org/source/
     * ftp://ftp.openssl.org/source/
