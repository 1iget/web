FIPS-140

For a basic introduction and some general background see Important Notes About OpenSSL and FIPS 140-2. For information about sponsorship and support, see OpenSSL and FIPS 140-2.

The most recent open source based validation of a cryptographic module (Module) compatible with the OpenSSL 1.0.1 and 1.0.2 libraries is v2.0.16, FIPS 140-2 certificate #1747. For convoluted bureaucratic reasons the same module is also available under the validations #2398 (revision 2.0.16) and #2473 (revision 2.0.10). This Module is documented in the 2.0 User Guide.

Thanks to multiple platform sponsorships the 2.0 validations include the largest number of formally tested platforms for any validated module.
